# HR Workforce Diversity & Attrition Dashboard

An interactive HR Analytics dashboard built using Tableau to analyze workforce diversity, employee attrition trends, employee satisfaction, and workforce planning metrics. This project helps HR teams and business leaders make data-driven decisions for employee retention, diversity & inclusion, and workforce optimization.

---

## 📌 Project Objective

The objective of this project is to create a professional, industry-oriented Tableau dashboard that tracks:

- Workforce diversity across departments
- Employee attrition patterns
- Employee satisfaction & work-life balance
- Promotion and overtime trends
- High attrition risk employee groups

This dashboard helps HR managers identify workforce issues early and improve employee engagement and retention strategies.
